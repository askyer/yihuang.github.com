最简单的语言 λ

    λx.axb == λy.ayb

    λx .f x == f

Simply Typed λ

定义：

如果a和b是类型，那么(a->b)也是类型。

e::a 表示表达式e的类型是a。

规则：

  1. 函数类型：当 e1::a 且 e2::b，那么 (λe1.e2)::a->b
  2. 调用规则：当 e1::a->b 且 e2::a，那么 (e1 e2)::b

Curry–Howard correspondence

(A ∧ B) ≡ (A, B)

(A ∨ B) ≡ Either A B

A ⇒ B ≡ A -> B

¬A ≡ A->undefined

Curry–Howard correspondence

A ⇒ A ∨ B
Left :: A -> Either A B

A ∧ B ⇒ A
fst :: (A, B) -> A

(A ⇒ (B ⇒ C)) ⇒ ((A ∧ B) ⇒ C)
proof :: (A -> (B -> C)) -> (A, B) -> C
proof f (a, b) = (f a) b

Typeclass - Ad-hoc poly

  • class Eq a where

  • instance Eq Int where
