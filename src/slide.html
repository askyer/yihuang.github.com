问题1

从列表取大于10且小于100的数

表达： 从列表取 ? 的数

λ x -> filter ? x

不如直接点：

filter ?

表达： 大于10

λ x -> x > 10

不如直接点：

(>10)

表达： 小于100

同样：

(<100)

表达： 且

&&

:: Bool -> Bool

类型不对

表达： 且

?

   (a -> Bool)
-> (a -> Bool)
-> (a -> Bool)

表达： 且

liftA2 (&&)

   (a -> Bool)
-> (a -> Bool)
-> (a -> Bool)

表达： 且

(&&&) = liftA2 (&&)

   (a -> Bool)
-> (a -> Bool)
-> (a -> Bool)

拼在一块：

filter ( (>10) &&& (<100) )

>>> let foo = filter ( (>10) &&& (<100) )
>>> foo [1..20]
[11, 12, 13 ... ]

Case study

正确答案： Maybe v

lookup :: k -> Map k v
       -> Maybe v

  • **type error**

What is Maybe

data Maybe a = Just a
              | Nothing

Maybe - 显式表达异常分支

  • fromMaybe :: a -> Maybe a -> a
fromMaybe _ (Just a) = a
fromMaybe a Nothing  = a

  • >>> fromMaybe 0
      (lookup k empty)

Monad - 重载语句

List Monad (list comprehension的马甲)

do a <- [1..10]
   b <- [1..10]
   guard $ a+b>10
   return (a, b)

-- [(1,10), (2,9), (2,10)...]
